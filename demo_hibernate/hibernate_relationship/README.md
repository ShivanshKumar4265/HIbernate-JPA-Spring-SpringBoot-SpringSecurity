# hibernate_relationship

A simple Hibernate + JPA console application for managing `Person` and `Aadhar` records.

## Menu options

- Add Person with Aadhar
- View all Persons
- View Person by ID
- Update Person
- Delete Person
- Add Aadhar
- View all Aadhars
- View Aadhar by Number
- Update Aadhar
- Delete Aadhar

## Run

Make sure MySQL is running and the schema in `src/main/resources/META-INF/persistence.xml` matches your local setup.

Then start the app from `org.jmd.App`.
